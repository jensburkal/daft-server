daft-server by colinmanning

Web application for Digital Asset Management access. Currently short on public documentation. This software is configured to work with the Cumulus 8.1 or later DAM system, and has an interface to support plugging in any DAM system that is Java accessable.

Uses a RESTful web service API to support integration of DAM solutions in enterprise environments, or as back ends to web portals. Provides connection pooling to optimise license usage, named and cached previews, automated asset and metadata upload. Programming interfaces for .NET, with high level bindings coming for Java and PHP. Sample code for simple web development using MooTools framework are available.
